# Второй мозг (память агента)

Эта папка — **долгая память** твоего агента, его «второй мозг».

## Зачем это нужно

Обычный чат всё забывает между разговорами. Агент с памятью — нет. Сюда он
записывает важные факты о тебе и о ваших делах, чтобы помнить их завтра, через
неделю и через месяц. Это превращает «бота, который отвечает» в «ассистента,
который тебя знает».

## Как это работает (просто)

- Агент сам пишет сюда заметки — например, «{{ВАШЕ_ИМЯ}} любит кофе без сахара»
  или «проект X сдать до пятницы».
- В следующий раз он сначала заглядывает в память, потом отвечает.
- Ты тоже можешь класть сюда файлы-заметки — агент их прочитает.

## Что положить сюда (задел под Урок 2)

Пока папка почти пустая — это нормально. В **Уроке 2** мы:
- научим агента записывать важное в память автоматически;
- заведём первые заметки о тебе.

> Файлы с твоими личными заметками здесь **не попадают** в публичный репозиторий
> (см. `.gitignore`) — память остаётся только у тебя на компьютере.
