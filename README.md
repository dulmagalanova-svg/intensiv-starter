# Твой личный AI-агент в Telegram — за час, без кода

Это готовая заготовка («стартер»). Ты её скачиваешь, подставляешь два своих
значения (токен бота и свой Telegram ID) — и у тебя свой персональный
AI-агент, который отвечает тебе в Telegram обычным языком. **Писать код не нужно.**

---

## Что это

Личный AI-ассистент, который:
- живёт у тебя на компьютере;
- отвечает тебе в Telegram, как живой собеседник (не бот по кнопкам);
- понимает обычный язык — пишешь словами, он понимает и отвечает.

Это фундамент. Дальше (Уроки 2–4) научим его быть «тобой», помнить контекст и
работать в команде.

---

## Что понадобится (5 минут на подготовку)

- **Компьютер** — Windows или Mac. Подойдёт твой личный (агент работает, пока
  компьютер включён — про «24/7» в конце).
- **Подписка Claude** — это «мозг» агента. Минимум **Pro ($20/мес)**, для уверенной
  ежедневной работы рекомендуем **Max ($100/мес)**.
- **Аккаунт Telegram.**
- **~1 час времени.**
- **Если ты в России** — включённый VPN, чтобы открывался сайт Claude.

> **Про деньги (важно и честно):** твой агент работает в обычной интерактивной
> сессии Claude Code под твоей подпиской **Pro/Max**. Это НЕ отдельный платный
> «Agent SDK» и не headless-режим — никакого отдельного счёта по API не появляется,
> расходуются обычные лимиты твоей подписки. Когда агент просто включён и ждёт —
> он не тратит ничего. Лимиты идут только когда он реально обрабатывает сообщения.

## Шаг 1. Установить Claude Code (15 мин)

Claude Code — программа, через которую агент «живёт» на компьютере. Ставится один раз.

1. Открой **Терминал** (Mac: программа «Терминал»; Windows: «PowerShell»).
   Это окно для команд. Мы даём готовые строки — копируй и жми Enter.
2. Установи Claude Code командой (нужен установленный Node.js 18+):

   ```bash
   npm install -g @anthropic-ai/claude-code
   ```

   > Нет Node.js? Поставь его с https://nodejs.org (кнопка LTS), затем повтори команду.

3. Войди в свой аккаунт Claude — запусти:

   ```bash
   claude
   ```

   Программа сама откроет вход через браузер (тут и нужен VPN, если ты в РФ).
   После входа можешь закрыть — главное, что вход выполнен.

✅ **Чекпоинт:** команда `claude` запускается и готова к работе.

---

## Шаг 2. Создать Telegram-бота (5 мин)

Бот — «лицо» агента в Telegram.

1. В Telegram найди **@BotFather** (официальный бот Telegram).
2. Отправь ему `/newbot`.
3. Придумай имя бота и его адрес (заканчивается на `bot`).
4. BotFather пришлёт **токен** — длинную строку вида `8976066018:AAH...`.
5. **Скопируй токен и никому не показывай** — это ключ от твоего бота, как пароль.

✅ **Чекпоинт:** токен бота у тебя на руках.

Способ А — командой (если ставил Claude Code, git обычно уже есть):

git clone https://github.com/dulmagalanova-svg/intensiv-starter.git intensiv-starter

✅ **Чекпоинт:** на компьютере лежит папка `intensiv-starter`.

## Шаг 4. Вставить токен и свой Telegram ID (5 мин)

1. Сначала узнай свой **Telegram ID** (это число, не телефон и не @username):
   в Telegram напиши боту **@userinfobot** — он пришлёт твой `Id`.
2. В папке `secrets/` скопируй файл `channel.env.example` и переименуй копию
   в `channel.env` (то же имя, без «.example»):

   ```bash
   cp secrets/channel.env.example secrets/channel.env
   ```

3. Открой `secrets/channel.env` обычным текстовым редактором и заполни:
   - `TELEGRAM_BOT_TOKEN=` — токен из Шага 2;
   - `TELEGRAM_EXPECTED_BOT_ID=` — цифры из токена **до двоеточия**;
   - `TELEGRAM_ALLOWED_USER_IDS=` и `TELEGRAM_ALLOWED_CHAT_IDS=` — твой Telegram ID из п.1.
4. Сохрани файл. Больше ничего трогать не нужно.

✅ **Чекпоинт:** в `secrets/channel.env` вставлены токен и твой ID.

> Безопасность: файл `channel.env` остаётся только у тебя на компьютере и никуда
> не выкладывается.

## Шаг 5. Установить зависимости (5 мин, один раз)

```bash
cd dashi-plugin-claude-code/plugin
bun install
cd ../..
```

> Нет команды `bun`? Установи Bun: `curl -fsSL https://bun.sh/install | bash`
> (на Mac можно `brew install bun`), затем повтори `bun install`.

✅ **Чекпоинт:** `bun install` завершился без красных ошибок.

---

## Шаг 6. Запустить агента (2 мин)

Из корневой папки `intensiv-starter`:

```bash
./start.sh
```

Агент «просыпается»: ты увидишь, что Claude Code запустился и слушает Telegram.
Это окно держи открытым — пока оно открыто, агент бодрствует.

✅ **Чекпоинт:** в окне видно, что агент запущен.

---

## Шаг 7. Написать боту — проверка (2 мин)

1. Открой Telegram, найди своего бота (по адресу из Шага 2).
2. Напиши ему: **«Привет, ты кто?»**
3. Через пару секунд агент отвечает сам.

🎉 **Готово. У тебя работает личный AI-агент в Telegram.**

## Если не работает (5 частых ошибок)

1. **Бот молчит.** Проверь, что окно из Шага 6 не закрыто (закрыл окно → агент
   уснул). И что в `channel.env` нет лишних пробелов в токене.
2. **«claude: command not found» / «bun: command not found».** Установка из Шага 1
   или Шага 5 не завершилась — повтори её, открой Терминал заново.
3. **Не открывается вход в Claude (Шаг 1).** Включи VPN (если ты в РФ) и попробуй снова.
4. **`start.sh` пишет «не найден secrets/channel.env».** Ты не скопировал
   `channel.env.example` → `channel.env` (Шаг 4, п.2).
5. **Бот отвечает «не тебе» или молчит на твои сообщения.** Проверь, что в
   `TELEGRAM_ALLOWED_USER_IDS` стоит именно твой ID из @userinfobot (число),
   а не @username и не номер телефона.

## Где держать агента

- **Локально на компьютере (по умолчанию).** Самый простой старт. Минус: агент
  работает, только пока компьютер включён и окно `start.sh` открыто.
- **Mac Mini как «домашний сервер».** Дешёвый тихий компьютер, который стоит
  включённым всегда. Ставишь туда тот же агент — он отвечает 24/7.
- **VPS (арендованный сервер) 24/7.** Для тех, кому нужно «всегда онлайн» без
  своего железа. Чуть сложнее в настройке (запуск как сервис), зато работает
  круглосуточно. Подробную настройку 24/7 разбираем на интенсиве.

## Сменить «мозг» модели

По умолчанию агент думает на **Claude** (твоя подписка). Можно переключить на
другую модель Claude (быстрее/дешевле) или на запасную модель через OpenRouter
(страховка от блокировок).

→ См. [docs/model-swap.md](docs/model-swap.md)
