# Как сменить «мозг» агента

> Задел под Урок 4 + страховка на случай блокировок.

«Мозг» агента — это модель, которая думает и пишет ответы. По умолчанию это
**Claude** (через твою подписку Claude Pro/Max). Для старта менять ничего не нужно —
всё уже работает на Claude.

Но иногда «мозг» хочется поменять:
- хочешь модель подешевле или побыстрее;
- хочешь не зависеть от одной подписки (страховка от блокировок/лимитов);
- в Уроке 4 мы разбираем «второй движок» отдельно.

---

## Вариант 1. Другая модель Claude (самое простое)

Claude бывает разный: посильнее и помедленнее (Opus), сбалансированный (Sonnet),
быстрый и дешёвый (Haiku). Переключение — одной строкой при запуске.

В файле `start.sh` найди строку запуска:

```bash
exec claude --dangerously-load-development-channels server:dashi-channel
```

Добавь `--model` с нужной моделью:

```bash
exec claude --model sonnet --dangerously-load-development-channels server:dashi-channel
```

- `--model sonnet` — быстрый и экономный по лимитам (рекомендуем для агента,
  который отвечает в течение дня);
- `--model opus` — самый «умный», но тяжелее по лимитам;
- без `--model` — берётся модель по умолчанию из твоей подписки.

Сохрани `start.sh`, перезапусти агента (`Ctrl+C`, потом снова `./start.sh`).

---

## Вариант 2. Совсем другая модель (через OpenRouter) — продвинутое

Это уже **не** Claude, а сторонняя модель (например, через сервис OpenRouter —
он даёт доступ к десяткам моделей по одному ключу). Подходит как страховка:
если доступ к Claude временно недоступен, агент продолжает работать на запасном «мозге».

Идея в одну строку: **указать Claude Code другой адрес API и ключ** через
переменные окружения перед запуском. Например, в `secrets/channel.env`:

```bash
# Запасной «мозг» через OpenRouter (пример — концептуально):
ANTHROPIC_BASE_URL=https://openrouter.ai/api/v1
ANTHROPIC_API_KEY=ваш_ключ_openrouter
```

> Это упрощённая схема для понимания принципа. Точную рабочую конфигурацию
> (какие именно переменные, какая модель, совместимость форматов) разбираем
> пошагово в **Уроке 4** — там же даём проверенный рецепт. Для Уроков 1–2
> оставайся на Claude (Вариант 1 хватает с запасом).

---

## Что выбрать сейчас

- **Уроки 1–2:** ничего не меняй — Claude по подписке.
- **Хочешь сэкономить лимиты:** `--model sonnet` (Вариант 1).
- **Нужна страховка от блокировок:** запасной «мозг» через OpenRouter (Вариант 2,
  подробно в Уроке 4).
