# Урок 1. Первый агент за час: запускаем — и он отвечает в Telegram

Личный AI-агент, который живёт у тебя на компьютере и отвечает тебе в Telegram —
как живой ассистент, а не бот по кнопкам. Ты пишешь ему обычным языком — он понимает
и отвечает. Это фундамент: дальше (Уроки 2–4) научим его быть «тобой», помнить
контекст и работать в команде.

**Гарантия урока:** к концу минимум один рабочий агент отвечает тебе в Telegram.

> Все команды ниже — реальные, из этой заготовки. Полная версия с чекпоинтами и
> разделом «если не работает» — в [README.md](../README.md) репозитория.

---

## Что понадобится (5 минут на подготовку)

- **Компьютер** — Windows или Mac (личный подойдёт; про «24/7» — в конце).
- **Подписка Claude** — «мозг» агента. Минимум **Pro ($20/мес)**, рекомендуем **Max ($100/мес)**.
- **Аккаунт Telegram.**
- **~1 час.**
- **Если ты в России** — включённый VPN, чтобы открывался сайт Claude.

---

## Шаг 1. Ставим Claude Code (15 мин)

Программа, через которую агент «живёт» на компьютере. Ставится один раз.

1. Открой **Терминал** (Mac: «Терминал»; Windows: «PowerShell»).
2. Установи Claude Code (нужен Node.js 18+ — если нет, поставь с https://nodejs.org, кнопка LTS):

   ```bash
   npm install -g @anthropic-ai/claude-code
   ```

3. Войди в аккаунт Claude:

   ```bash
   claude
   ```

   Откроется вход через браузер (тут нужен VPN, если ты в РФ).

✅ Команда `claude` запускается — инструмент стоит.

---

## Шаг 2. Создаём Telegram-бота (5 мин)

1. В Telegram найди **@BotFather**.
2. Отправь `/newbot`.
3. Придумай имя и адрес бота (заканчивается на `bot`).
4. Скопируй **токен** вида `8976066018:AAH...` — это пароль от бота, никому не показывай.

✅ Токен бота у тебя на руках.

---

## Шаг 3. Скачиваем заготовку (5 мин)

```bash
git clone https://github.com/dulmagalanova-svg/intensiv-starter.git intensiv-starter
cd intensiv-starter
```

Или кнопкой **Code → Download ZIP** на странице репозитория → распакуй → открой папку.

✅ Папка `intensiv-starter` на компьютере.

---

## Шаг 4. Вставляем токен и свой Telegram ID (5 мин)

1. Узнай свой **Telegram ID**: напиши в Telegram боту **@userinfobot** — он пришлёт твой `Id` (число).
2. Скопируй файл настроек и убери «.example»:

   ```bash
   cp secrets/channel.env.example secrets/channel.env
   ```

3. Открой `secrets/channel.env` в текстовом редакторе и заполни:
   - `TELEGRAM_BOT_TOKEN=` — токен из Шага 2;
   - `TELEGRAM_EXPECTED_BOT_ID=` — цифры токена **до двоеточия**;
   - `TELEGRAM_ALLOWED_USER_IDS=` и `TELEGRAM_ALLOWED_CHAT_IDS=` — твой ID из п.1.
4. Сохрани.

✅ Токен и ID вставлены.

---

## Шаг 5. Ставим зависимости (5 мин, один раз)

```bash
cd dashi-plugin-claude-code/plugin
bun install
cd ../..
```

> Нет `bun`? Установи: `curl -fsSL https://bun.sh/install | bash`, повтори `bun install`.

✅ `bun install` прошёл без ошибок.

---

## Шаг 6. Запускаем агента (2 мин)

Из корня `intensiv-starter`:

```bash
./start.sh
```

Окно держи открытым — пока открыто, агент бодрствует.

✅ В окне видно, что агент запущен и слушает Telegram.

---

## Шаг 7. Пишем боту — проверка (2 мин)

1. Найди своего бота в Telegram.
2. Напиши: **«Привет, ты кто?»**
3. Через пару секунд агент отвечает сам.

🎉 **Готово. Личный AI-агент в Telegram работает.**

---

## Если что-то не работает

См. раздел «Если не работает (5 частых ошибок)» в [README.md](../README.md).
Коротко: чаще всего — закрыто окно `start.sh`, не скопирован `channel.env`,
выключен VPN, или в ID попал @username вместо числа.

---

## Сколько это стоит (честно)

- Агент включён и ждёт — **не тратит ничего.** Лимиты идут только на обработку сообщений.
- Работает в обычной интерактивной сессии Claude Code под подпиской **Pro/Max** —
  это НЕ отдельный платный Agent SDK, отдельного счёта по API не возникает.
- **Pro ($20)** — для небольшого объёма. **Max ($100)** — для уверенной ежедневной работы.
- Узкое место — не «сколько часов работает», а **сколько сообщений в день** обрабатывает.
